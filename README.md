# Social Media App

Simple full-stack starter with:
- `frontend/`: React + Vite + Tailwind CSS
- `backend/`: Express + MongoDB + Mongoose

## Setup

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

Set `MONGO_URI` in `backend/.env` before starting the backend.

## Project Structure

```text
social-media/
  frontend/
  backend/
```
